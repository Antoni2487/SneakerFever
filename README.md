# SneakerFever
Spring Boot + Thymeleaf | Proyecto académico UTP

Tienda online de zapatillas, ropa y accesorios con marcas reconocidas.
Proyecto académico desarrollado en la Universidad Tecnológica del Perú (UTP).

## Tecnologías

- Java Spring Boot
- Thymeleaf
- MySQL
- Bootstrap 5
- HTML / CSS

## Funcionalidades

- Catálogo de productos por categorías (zapatillas, ropa, accesorios)
- Vistas dinámicas con Thymeleaf
- Diseño responsivo para móvil y desktop
- Organización de productos por marca

## Cómo correrlo localmente

1. Clona el repositorio
   git clone https://github.com/TuUsuario/SneakerFever.git

2. Configura la base de datos en application.properties
   spring.datasource.url=jdbc:mysql://localhost:3306/sneakerfever
   spring.datasource.username=tu_usuario
   spring.datasource.password=tu_password

3. Ejecuta el proyecto desde tu IDE (IntelliJ o Eclipse)
